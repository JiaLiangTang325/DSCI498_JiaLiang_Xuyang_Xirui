# DSCI498_JiaLiang_Xuyang_Xirui
# Image Style Transfer Project

## Introduction

This project implements image style transfer using deep learning techniques.
The goal is to generate artistic images by combining the **content of one image** with the **style of another image**.

The project includes preprocessing, style transfer implementation, and batch image generation.
All generated results are stored in the `results` folder.

---

## Project Structure

```
project/
│
├── data/                     # Input images used for style transfer
├── results/                  # Generated images
│
├── 01_preprocess.py          # Image preprocessing script
├── 02_style_transfer.py      # Core style transfer implementation
├── 03_batch_style_transfer.py# Batch generation of stylized images
│
└── README.md                 # Project documentation
```

---

## Requirements

The project requires the following environment:

* Python 3.x
* PyTorch
* NumPy
* scikit-image
* Matplotlib

Install dependencies using:

```
pip install -r requirements.txt
```

---

## How to Run

Step 1: Prepare input images in the `data` folder.

Step 2: Run the batch style transfer script:

```
python 03_batch_style_transfer.py
```

Step 3: The generated stylized images will be saved in the `results` folder.

---

## Results

The `results` folder contains the generated images from the batch style transfer process.

Each output image is generated by combining a **content image** with a **style image**.

The naming format of the images typically follows:

```
contentname_stylename.png
```

This format indicates which content image and style image were used to generate the result.

---

## Notes

Some generated images may appear darker or different due to variations in style intensity or preprocessing settings.

All outputs are generated automatically by the batch style transfer script.
